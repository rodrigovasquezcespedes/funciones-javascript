Ejercicio 1

Transforma esta declaración de función a una función de expresión.

function example(a, b, c)
{ return a+b+c }

Respuesta

const example = function (a, b, c)
{ return a + b + c; }

Ejercicio 2

Transforma la siguiente función a una arrow function de una línea.

let suma = function(a, b)
{ return a + b }

respuesta

let suma = (a, b) => a + b;
